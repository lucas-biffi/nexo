# Nexo - Fintech Digital Bancária

## Sobre o Projeto
Este projeto foi proposto na universidade e consiste na criação de uma fintech de banco digital voltada para oferecer serviços financeiros simples, acessíveis e seguros. A proposta é atender usuários que buscam praticidade no dia a dia, sem a complexidade de produtos bancários avançados.

## Funcionalidades
- Cadastro de usuários;
- Login;
- Criação automática de conta ao finalizar o cadastro; 
- Transferência pix;
- Pagamento de boleto;
- Visualização de saldo;
- Visualização de extrato;
- Tela inicial;
- Tela de cotação de dólar e euro.

## Tecnologias Utilizadas
- **Backend:** Node.js / Express / MySQL / JWT
- **Frontend:** Html / CSS / JavaScript  
- **Banco de Dados:** MySQL Workbench
